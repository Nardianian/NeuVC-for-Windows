# NeuVC
Neural based vst plugin for VoiceConversion 

# install 
put JUCE folder on the Thirdparty folder, then run
```
$ mkdir build 
$ cd build 
$ cmake .. 
```

## important (Env setup) 

you need to move the whole (with models downloaded) RVC repo in NeuVC working directory (on Mac that's on ..user_name/Library/NeuVC )
